The %APPDATA% variable is defined by Microsoft Windows to refer to your personal application data. For example, in Windows 7

expands to

C:\Users\username\AppData\Roaming\FlareLint

where username is your user account on your computer.

The file names for rules end with rule.txt. The files for default rules have names starting with flarelint_.

Tip: If you create your own rules, consider using file names that start with something descriptive, like the name of your team or company.
