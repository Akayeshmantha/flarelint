next

next tag

True when the sibling that immediately follows an element is named tag, false otherwise.

This predicate can be used as a point of view.

For example:

<h1>New Brunswick Bucket List</h1>
<p>Never negate the boundless beauty of these places:</p>
<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the h1 element, the following predicates give true:

next p
after p
after ul

For the h1 element, the following predicate gives false:

next ul
