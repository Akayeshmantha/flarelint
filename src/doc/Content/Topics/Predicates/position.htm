pov-position

pov-position tag number
pov-position * number

True when an element tag is the nth child among its siblings. The sibling elements do not have to be the same. The first child is at position 0.

This predicate ignores descendants of children. For example:

<body>
  <h1>Cities with Airport Train Stations</h1>
  <p>The best way to get to an airport is by train:</p>
  <ul>
    <li>Barcelona</li>
    <li>Paris</li>
    <li>New York</li>
  </ul>
</body>

For the body element, the following gives true:

child-position ul 2

For the p element, the following gives true:

after-position ul 2

For the li element containing “New York”, the following gives true:

self-position li 2

For the li element containing “Barcelona”, the following gives false:

self-position li 0
