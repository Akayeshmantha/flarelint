pov-last

pov-last tag
pov-last *

True when an element tag is the last child among its siblings. The sibling elements do not have to be the same.

This predicate ignores descendants of children. For example:

<body>
  <h1>Cities with Airport Train Stations</h1>
  <p>The best way to get to an airport is by train:</p>
  <ul>
    <li>Barcelona</li>
    <li>Paris</li>
    <li>New York</li>
  </ul>
</body>

For the body element, the following gives true:

child-last ul

For the p element, the following gives true:

after-last ul

For the li element containing “New York”, the following gives true:

self-last li

For the li element containing “Barcelona”, the following give false:

self-last li
after-last li
next-last li
