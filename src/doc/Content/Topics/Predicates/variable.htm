pov-variable

pov-variable tag set.name
pov-variable * set.name

True when an element contains at least one reference to a variable, set.name, false otherwise.

The variable must use the format of variable set, period (.), then a name defined in the variable set. For example, to specify the system (set “System”) chapter number (name “ChapterNumber”) variable, use System.ChapterNumber.

For example:

<p>
  <MadCap:variable name="Acme.CEO" /> 
  ran madly off in all directions.
</p>

For the p element, the following gives true:

self-variable p Acme.CEO
