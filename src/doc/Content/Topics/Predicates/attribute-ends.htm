pov-attribute-ends

pov-attribute-ends tag name "text"
pov-attribute-ends * name "text"

True when an element has an attribute name and its value ends with text. False when either an element does not have an attribute name or, if it does have this attribute, its value does not end with text.

This predicate is case-sensitive. Leading and ending spaces and line endings in the attribute value are ignored.

For example:

<a href="#Fredericton ">Home of Bliss Carman</a>

For the a element, the following give true:

self-attribute-equals a href "ton"
self-attribute-equals * href "#Fredericton"

And the following give false:

self-attribute-equals a href "#Fred"
self-attribute-equals * href "TON"
