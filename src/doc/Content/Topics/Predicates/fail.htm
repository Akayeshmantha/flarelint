fail

Always false. It is useful for rule tests, when an element match is sufficient to determine that a rule is broken and needs no further tests.

For example, the following rule results in an error message when an h3 element appears in a topic:

rule: error
extensions: htm html
when: self h3
test: fail
message:
Unsupported header. This level of header is not supported by our
conventions or styles. To fix, place a subtopic in a separate
file.
