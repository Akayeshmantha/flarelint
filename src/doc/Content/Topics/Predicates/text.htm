pov-text

pov-text tag "text"
pov-text * "text"

True when an element, excluding the content of its descendant elements, contains text. False when text does not appear in an element.

The text is case sensitive.

For example:

<p><i>Lord Ronald</i> ran madly off in all directions.</p>

For the p element, the following give true:

not self-text p "Ronald ran"
child-text i "Ronald"
self-text p "madly off in all directions"

For the p element, the following give false:

not self-text p "RONALD ran"
child-text i "ron"
self-text p "Madly Off In All Directions"
