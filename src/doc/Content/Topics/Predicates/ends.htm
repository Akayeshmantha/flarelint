pov-ends

pov-ends tag "text"
pov-ends * "text"

True when an element, including the content of its descendant elements, ends with text. False when text does not appear at the end of an element and its descendants.

This predicate is case sensitive. It ignores following white space and line breaks.

For example:

<p>
  <i>Lord Ronald</i> ran madly off in all <b>directions</b>!
</p>

For the p element, the following give true:

self-ends p "all directions!"
child-ends b "directions"
