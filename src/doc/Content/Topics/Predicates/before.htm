before

before tag
before *

Is true when an element has a preceding sibling element with tag tag, false otherwise.

For example:

<h1>New Brunswick Bucket List</h1>
<p>Never negate the boundless beauty of these places:</p>
<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the ul element, the following predicates give true:

before h1
before p

When used as a point of view, before applies to the first sibling named tag.

For example:

<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the li element containing “Kouchibouguac”, the following gives true:

before-contains li "Saint"

But the following gives false:

before-contains li "Edmun"

Even though there is an li element before “Kouchibouguac” that contains “Edmun”, the latter is not the first li element before “Kouchibouguac”.
