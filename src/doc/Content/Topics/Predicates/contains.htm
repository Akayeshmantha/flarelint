pov-contains

pov-contains tag "text"
pov-contains * "text"

True when an element, including the content of its descendant elements, contains text. False when text does not appear in an element and its descendants.

The text is case sensitive.

For example:

<p><i>Lord Ronald</i> ran madly off in all directions.</p>

For the p element, the following predicates give true:

self-contains p "Ronald ran"
child-contains i "Ron"
self-contains p "madly off in all directions"

For the p element, the following predicates give false:

self-contains p "RONALD ran"
child-contains i "ron"
self-contains p "Madly Off In All Directions"
