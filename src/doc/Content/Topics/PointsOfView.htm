Point of View

Most predicates start with a point of view. A point of view specifies where to look relative to an element.

For example:

<html>
  <body>
    <h1><em>Cats</em> Create Chaos</h1>
    <p>Top cat chaos activities:</p>
    <ol class="CatList">
      <li>purring</li>
      <li>shredding upholstery</li>
      <li>meowing at <b>night</b></li>
    </ol>
    <h2>Kittens</h2>
    <p>Kittens are not chaotic.</p>
  </body>
</html>

For the element ol, the following predicates are true:

after h2
after p
ancestor body
ancestor html
before h1
before p
child li
descendant b
next h2
parent body
previous p
self ol

And the following predicates are false:

after body
ancestor h1
before h2
child b
descendant p
next p
parent html
previous h2
self p

You may specify a predicate with just a pov or you may optionally extend the predicate to be more specific.

For example, for ol the following predicates are true:

self-attribute-equals ol class "CatList"
child-begins li "shredding"
before-position h1 0
