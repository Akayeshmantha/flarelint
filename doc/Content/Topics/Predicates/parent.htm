parent tag
parent *

True when the element that immediately contains an element is named tag, false otherwise.

For example:

<h1>New Brunswick Bucket List</h1>
<p>Never negate the boundless beauty of these places:</p>
<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For any li element, the following predicate gives true:

parent ul
