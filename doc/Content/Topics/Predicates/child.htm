child

child tag
child *

Is true when an element has a child with tag tag, false otherwise.

For example:

<h1>New Brunswick Bucket List</h1>
<p>Never negate the boundless beauty of these places:</p>
<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the ul element, the following predicate gives true:

child li

When used as a point of view, child applies to the first child named tag.

For example:

<ul>
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the ul element, the following gives true:

child-contains li "Edmun"

But the following gives false:

child-contains li "Saint"

Even though there is a child li element that contains “Saint”, it is not the first child li element.
