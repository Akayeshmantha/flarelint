language

language tag

True when the element’s language attribute, xml:lang, begins with tag. This predicate is false otherwise.

If the element has no xml:lang attribute, then examine the nearest ancestor that has an xml:lang attribute. If there is no xml:lang attribute in the Flare file being scanned, then use the language specified in the Flare project’s properties. If the Flare project does not specify a language, then use the Flare default, en-us.

This predicate is case-insensitive.

For more information on XML language tags, see Language tags in HTML and XML.
