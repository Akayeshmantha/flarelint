Uninstalling

To remove and your rules from your computer, you need to uninstall the Python package, uninstall Python, then delete your rules.

To uninstall :

  1. Open the Uninstall or Change a Program control panel.

    In the Windows Start menu, click Control Panel. Under Programs, click Uninstall a Program.

  2. Double-click Python -x, where x is the version.

  3. In the Programs and Features alert, click Yes to confirm that you are sure you want to uninstall .

  4. In the Uninstall or Change a Program window, double-click the Python application.

    For example, if you installed Python 3.5.2, then double-click Python 3.5.2 (64-bit).

  5. Click Uninstall.

  6. In the Uninstall or Change a Program window, double-click Python Launcher.

  7. In the Programs and Features alert, click Yes to confirm.

  8. If the User Account Control alert appears to prompt you to allow the installer to change your computer, click Yes.

  9. Click the Windows Start menu then type %APPDATA% and press Enter.

    A Windows Explorer window appears to display your personal data folder.

  10. Select the FlareLint folder by clikcing it then delete it by pressing the Delete key to put it in the Recycle Bin.
