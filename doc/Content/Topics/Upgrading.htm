Upgrading

To upgrade on your computer, you need to uninstall the current Python package then install your new Python package.

Note: If you are upgrading from 1.1 or 1.2 to 1.3 or later, you need to also need to re-create any custom rules your team uses. For more information, see Creating and Customizing Rules.

To upgrade :

  1. Download and extract the contents of the .zip installer for the new tool.

    www.intelerad.com/flarelint

    Tip: Unzip .zip into a folder on your desktop.

  2. Open the Uninstall or Change a Program control panel.

    In the Windows Start menu, click Control Panel. Under Programs, click Uninstall a Program.

  3. Double-click Python -x, where x is the version of the currently-installed .

  4. In the Programs and Features alert, click Yes to confirm that you are sure you want to uninstall the old version of .

  5. In the unzipped folder, double-click the .msi file.

  6. In the Python Setup wizard, enable Install Just for Me and click Next.

  7. Choose Python x from Registry and click Next.

    The installation begins.

  8. If the User Account Control alert appears to prompt you to allow the installer to change your computer, click Yes.

  9. When the wizard is done, click Finish.
