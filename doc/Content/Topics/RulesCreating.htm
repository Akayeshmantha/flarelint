Implementing and Modifying Rules

You can modify existing rules and implement new ones. rules are defined in Python modules. Each rule instantiates an object of class Error or Warning. These classes are defined in the flarelint.rule module of the package. A rule can inspect the contents of a Flare file through the use of a Node object, defined in the flarelint.flarenode module.

For detailed information on rules and Flare nodes, see their respective source files in the package.
