Using

scans your MadCap Flare project for conformance to your rules. scans topics, snippets, TOCs, and any other XML-based files in the project.

It then generates a report named Report.html and opens this report in your default web browser.

If there is an existing report in your Flare project folder, renames the older report by appending a number. You can copy, move, and delete reports without affecting your Flare project.

Before using , make sure you and your colleagues have saved changes before running . scans only saved files. It cannot read unsaved changes.

To read the report later, look for it in the same folder as your project. The file name of the most recent report is Report.html.

To use :

  1. In Windows Explorer go to the folder for your Flare project.

  2. Drag the project file (.flprj) and drop it on .bat.

    Note: Before dropping the project file, make sure the "Open with .bat" tool tip appears.

  3. A Windows Security alert might appear. If so, click Yes.

    A Windows command window opens to show progress then closes if there is no error.

    opens the report in your web browser.

  4. If the follow appears in the command window, you must re-install Python. See Installing.

    The error message 'python' is not recognized... means that you did not enable the Add Python x to PATH checkbox in the Python installer.

  5. In your browser, read the report.
